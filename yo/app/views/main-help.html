Logging in

When you first access TopCAT, you will be presented with a login page.

You can login by entering your username and password provided by the Diamond User Office. Alternatively, to view only public data, select Anonymous from the Authentication Type drop down menu anc click login.

Once successfully logged in, you will see a page with four tabs.

My Data

The My Data tab gives you quick access to all your investigations and is presented in a grid.

Sorting and Filtering

Investigatons can be sorted by clicking on the column title. If a column has an input box, you can use it to filter results. Wildcard filter is possible using the character * to match multiple characters or ? to match a single character. For Date filters, dates must be entered in the format YYYY-MM-DD. Both From.. and To... dates must be entered.

Meta Data

If you click on a row (make sure it is not a hyperlink), tabs will appear on the bottom half of the page. Each tab gives you more information about the investigation.

Adding and removing items from the cart

On the very first column of each row on the grid, you will see a tick box.

Clicking this box will add or remove the selected investigation to your Cart. Multiple items can be added using the "Shift" key.

Clicking on a link will take you to the datasets of that investigation.

You can use the Cart items counter on the top right corner to check if an item was added or removed.

Browse

The Browse tab allows you to browse your data and any public data.

The grid works the same as the grid in the My Data tab. You can sort and filter. You can drill drown the data hierarchy using the hyperlinks in each row. You can also use the breadcrumb to go up the hierarchy. The breadcrumb is useful to tell where you are in the data hierarchy.

If the first column of the gird is a tick box, you can click the box to add the item to your cart. Please note if you select an investigation, all dataset and datafile belonging to that investigation will be selected. It is not possible for example to unselect one datafile if the parent investigation is already selected.

Similarly to My Data, clicking on a row (not hyperlink) will display it's meta data information in the panel at the bottom of the page.

Cart

The Cart tab is a grid that list all the items that you have added to the cart.

Here you can remove already added items or request download of the item by Clicking the Download Cart button. When the button is clicked, you will see a Download modal window and be presented with serveral options.

The Download Name is a name that identify the particular download. It is also used as the filename when your download becomes available.

The Transport selection option determines how you want to download the file. The options are as follows:

Transport Name Description Additional Signup/Software
Https Your download will be done via https via your browser or other tools suc as wget, curl etc. None
Globus Your download will be prepared so you can initial a Globus ftp download (Better description from Frazer needed) Signup at https://www.globus.org/SignUp
Globus Connect Personal client is required
Smartclient Your files are downloaded individually to your machine via the Smart client (more details). The Smartclient option will only appear if the Smart Client is already on you machine. Smart Client is required

You can optionally enter your email. When your download is available, an email will be sent.

Clicking the Ok button on the modal will submit your download request to be process and you should see a Cart successfully submitted notification. The request will be added to the Download Order list in the Download tab.

Download

The Download tab is a grid that list all your download requests.

The Status column gives you the current status of your download. Possible values are RESTORING and COMPLETE. The Download button is disabled when the status is restoring. Once its status becomes complete, you can click the download button to download the file via the browser if Https transport type was used. Alternatively, use can right click the button and copy the link address and use tools such as wget or curl to download the file.

For download using Smartclient as the tranport type, your files are downloaded you your machine as soon as is restored from tape archive. The complete status for smartClient download means all files were downloaded to your machine. Since smartclient download files individually rather than a zip archive, you can access already downloaded files in your Smart Client home directory even though the entire download has not yet completed.

For download using Globus as the transport type, when the status is complete for the download, you can login to Globus Online and initial the download to your endpoint.
