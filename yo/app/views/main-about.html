TopCAT allows you to browse and download archived data from beamline experiments at Diamond Light Source.

How to use TopCAT

Logging in

When you first access TopCAT, you will be presented with a login page.

You can login by entering your the username and password provided by diamond. Alternatively, to view only public data, select Anonymous from the Authentication Type drop down menu anc click login.

Once logged in, you will see four tabs

My Data

The My Data tab gives you quick access to all your investigations and is presented in a grid. Investigatons can be sorted by clicking on the column title. If a column has an input box, you can use it to filter results. Wildcard filter is possible using the character '%''. Dates must be entered in the format YYYY-MM-DD.

If you click on a row (make sure it is not a hyperlink), tabs will appear on the bottom half of the page. Each tab gives you more information about the investigation.

On the very first column of each row on the grid, you will see a tick box. Clicking this box will add or remove the selected investigation to your Cart. You can use the number of items indicator on the top right corner to see if an item was added or removed. Multiple items can be added using the "Shift" key.
